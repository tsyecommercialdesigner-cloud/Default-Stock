基于 Chromium 的那类浏览器，通常可以用 Chrome 的内置开发者工具来做整页长截图：
先打开开发者工具，再执行“Capture full size screenshot”之类的命令即可导出整页图片。

## 电脑端做法

1. 打开你要截取的网页。
    
2. 按 **Ctrl + Shift + I** 打开开发者工具。
    
3. 再按 **Ctrl + Shift + P** 打开命令面板。
    
4. 输入 **capture** 或 **full**，选择 **Capture full size screenshot**。
    
5. 浏览器会自动保存整页长截图，通常是 PNG 文件。
    

## 备用方案

- 如果浏览器自带截图工具里有“整页截图”“滚动截图”“Full page”选项，也可以直接用那个功能。
    
- 如果没有内置长截图功能，可以安装支持滚动截屏的扩展，例如**GoFullPage**这类的插件。
    
- 也可以试 **Awesome Screenshot** 或其他滚动截图插件，这类工具对长网页通常更兼容。
	

## 常见问题

- 如果按 **Ctrl + Shift + P** 直接弹出打印窗口，先确认已经打开了开发者工具，再按这个快捷键。
    
- 如果页面里有图片、评论、商品列表这种懒加载内容，最好先完整滚动一次，否则截图命令只能拿到已加载部分。
    
- 如果是某些特殊页面，可能只能用“分段截图 + 拼接”的方式解决，因为页面结构本身不支持完整整页抓取。
