**重点参数**

编辑 `/opt/dify/docker/.env`：

```
cd /opt/dify/docker
sudo nano .env
```

设置为：

```
# 腾讯云 CVM 使用国内 PyPI 镜像
PIP_MIRROR_URL=https://mirrors.cloud.tencent.com/pypi/simple

# 防止依赖下载较慢时被 daemon 提前终止
PLUGIN_PYTHON_ENV_INIT_TIMEOUT=900
PLUGIN_MAX_EXECUTION_TIMEOUT=1200
```

保存后退出：`Ctrl+O`、回车、`Ctrl+X`。

`PIP_MIRROR_URL` 是核心：避免直接访问 PyPI 导致依赖安装过慢；后两个参数用于放宽插件 Python 环境初始化及执行的超时时间。Dify 1.15 的 Compose 会读取这两个变量并传给 `plugin_daemon`。[Dify 1.15 Compose 配置](https://raw.githubusercontent.com/langgenius/dify/1.15.0/docker/docker-compose.yaml) 腾讯云官方 PyPI 镜像地址为上述 URL。
