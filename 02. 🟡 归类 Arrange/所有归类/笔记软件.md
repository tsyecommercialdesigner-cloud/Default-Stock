# 我的需求
- 能够多端同步
- 本地储存
- 能够和AI很好协同，写的笔记AI能读懂

# 参考中的软件
- Notion
	- 优点
		- 操作直观，不需要过多学习成本
		- 自动化功能
		- 协作
	- 缺点
		- 需要科学上网
		- 需要自己搭建系统，比较花时间
- Obsidian
	- 优点
		- 双链
		- 图谱
		- 百般
	- 缺点
		- 不符合word排版
		- 功能比较多，需要花时间学习
		- 必须手动添加笔记之间的关联，否则找不到
- 思源
	- 优点
		- 基本功能都有
		- 排版简单
	- 缺点
		- 没有白板
		- 数据库视图比较少
		- 不是md格式储存
- Fabric
	- 优点
		- 可以把pdf、图片、文字作为笔记
		- 能够用对话方式，直接向ai提问，找到对应笔记
	- 缺点
		- 无法导出markdown
		- 功能固定
		- 同步附件需要付费
		- 费用比较高
			- $5/month
