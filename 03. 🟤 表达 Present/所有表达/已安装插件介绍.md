一个是在左下角设置 - 选项 - 第三方插件
这里是所有安装的插件

## 已安装的 款插件简介
具体插件的使用查看视频教程

### clear unused images
帮忙删除没有在笔记中使用的图片附件
直接点击左侧菜单栏，有一个长得像图片的按钮
点击之后会自动删除
### commander
让你在各个位置添加自己想要的按钮
比如这个仓库右上角的加入标签、分屏、置顶都是用commander这个插件来实现的


### Day Planner
能够可视化显示一天当中日程的插件
搭配日记模版一起使用，在右上角有按钮可以点击查看

### Douban
可以快速帮我们提取书、电影电视局的基本信息，记录在笔记里面
很适合我们用obsidian来打造自己的图书馆
我已经做好了设置，你用豆瓣倒入的书、影片会到相应的文件夹当中

### file explorer note count
文件夹里面笔记的数量

### Hover editor
能够预览链接笔记，并且对预览的笔记进行管理

### journals
用来做百年日记系统的插件，包括了这个日历 日记页面上的导航栏
非常好用的日记管理插件

### media extended
用来边看视频边记笔记 可以记录时间轴的工具
使用的时候先用ctrl/cmd p 搜索media extended
输入视频的地址，就能边看视频 边记笔记了
还能打上时间戳

### Note Refactor
能帮你把选中的内容再单独创建一个新的笔记
我给大家设置了快捷键
我们来快捷键里面看看 note refactor
生成的内容
### outliner
可以拉动大纲
同时有快捷键来收缩大纲列表
快捷键可以看这里[[常用快捷键#大纲模式下使用快捷键]]

### Pandoc Plugin
按照教程配置好之后，可以直接导出word，ppt等更多格式的内容
具体教程可以参考这个视频
https://www.bilibili.com/video/BV19B76z8E7e/
### Paste URL into selection
快速方便的把超链接加到选中的文字上面
选中一段文字之后，再粘贴URL，这段文字就变成超连接了

### PDF++
非常好用的PDF批注软件
基础教程 https://www.bilibili.com/video/BV1zR28Y6EMQ/
进阶教程在系统课

### PKMer
登录之后，就能无需网络下载各种第三方插件了
当然如果你可以翻墙的话，不需要这个软件
第三方插件的使用方法查看对应课程
### remotely save
实现苹果移动端和windows电脑同步的小工具
点击左侧菜单栏上的对应按钮，就能手动同步

### Strange New Worlds
自动在每个超链接右上角显示它被链接了多少次

### Tags overview
能够看到标签下的所有笔记
### templater
可以把代码储存到模版当中，用它创建笔记的时候能实现更多效果
我是直接用它取代官方的模版插件来使用

### TimeStamper
方便快速记录现在的时间，搭配day planner来用
我设置了快捷键，这样可以快速记录自己那个时间在做什么事情，直接生成时间轴了
